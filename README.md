### [3] Introduction
Hit Me Up is a full stack web application built using Next.js, Socket.io, Node.js and other cool technologies and APIs. It is a social media app and the main tech in this project is Socket.IO as it is used to create bidirectional communication between users to implement chat and notifications. It has all the typical features you'd see in a social media such as resetting password, uploading a pic (even cropping them before uploading), commenting, liking, and so on. All the features aren't discussed in this README as it will get long and complicated. Check it out to learn more.

### [4] Technologies Used/Features Implemented
* Axios
* Express.js
* JWT
* Mongoose
* Next.js
* SendGrid API (for password reset feature where link to reset password would be mailed to users who forgot their passwords)
* React
* Infinite scroll feature
* Image cropping feature before uploading post
* Popup notifications
* Semantic UI React
* Socket.IO
